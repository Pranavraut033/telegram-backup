# Contributing

Thanks for your interest in improving Telegram Backup.

## Development setup

1. Create and activate a virtual environment.
2. Install dependencies with `pip install -r requirements.txt`.
3. Run the CLI locally with `python main.py --help`.

## Pull request guidelines

- Keep PRs focused and small.
- Describe user impact and testing done.
- Update docs when behavior changes.
- Do not commit secrets, `.env`, or session/state artifacts.

## Reporting issues

When filing an issue, include:

- Platform (macOS/Linux/Windows)
- Python version (if running from source)
- Command used and observed output
- Steps to reproduce
